= Test_ELog =
Test event logging for Arduino Events Library and Nokia 5110 LCD




== Nokia LCD Pinout ==
<pre>
Arduino Uno       Nokia 5110 Display
3.3V -------------------- 1-VCC
PIN #7 ------------------ 3-SCE
PIN #6 ------------------ 4-RST
PIN #5 ------------------ 5-D/C
PIN #4 ------------------ 6-DNK(MOSI) (SDIN)
PIN #3 ------------------ 7-SCLK
</pre>

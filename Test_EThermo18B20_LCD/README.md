= Test_ELog =
Test Thermometer logging for Arduino Events Library and Nokia 5110 LCD




== Nokia LCD Pinout ==
<pre>
Arduino        Nokia 5110 Display
3.3V   ------------------ 1-VCC
Ground ------------------ 3-SCE
Reset  ------------------ 4-RST
PIN #5 ----R10k---------- 5-D/C
PIN #4 ----R10k---------- 6-DNK(MOSI) (SDIN)
PIN #3 ----R10k---------- 7-SCLK

</pre>
